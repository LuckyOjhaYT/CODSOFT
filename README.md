# Java Programming Tasks - CODSOFT

This repository contains my solutions for the Java Programming internship tasks at CODSOFT. These console-based applications demonstrate my understanding of Core Java, Object-Oriented Programming (OOPs), and basic algorithmic logic.

## 🛠️ Tech Stack
- **Language:** Java (JDK 8+)
- **Concepts:** OOPs (Encapsulation, Classes/Objects), Control Structures, Loops, `java.util.Scanner`

## 📁 Projects Included

### 1. ATM Interface (`Main.java`)
A console-based ATM system that allows users to deposit, withdraw, and check their balance.
* **Key Features:** Input validation, insufficient balance handling, and a continuous transaction menu.
* **Concepts Used:** Object-Oriented Programming, Separation of Concerns (BankAccount vs ATM class), `switch-case`.

### 2. Number Game (`NumberGame.java`)
An interactive guessing game where the user has to guess a randomly generated number within a limited number of attempts.
* **Key Features:** Score tracking, multiple rounds, and feedback (too high/too low).
* **Concepts Used:** `Math.random()`, `while` loops, conditional statements.

### 3. Student Grade Calculator (`StudentGradeCalculator.java`)
A utility application that takes marks obtained in different subjects and calculates the total marks, average percentage, and the corresponding grade.
* **Key Features:** Dynamic input taking and structured grade calculation logic.
* **Concepts Used:** Arrays (optional), basic arithmetic logic, and control flow.
